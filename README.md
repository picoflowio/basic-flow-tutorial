# picoflow.io — Basic Flow Tutorial

Static Eleventy site that teaches how to build Picoflow’s `BasicFlow` step-by-step. The content lives in the repo root (`*.md`, `app.mjs`, `app.styles.scss`) and the site engine is under `.app/`.

## Run locally
- `cd basic-flow-tutorial/.app`
- `npm install`
- `npm run start` (serves at http://localhost:8080)

## Build
- `cd basic-flow-tutorial/.app`
- `npm run build` (outputs to `.app/dist`)

## Content map
- Lessons: `basic-flow-tutorial/*.md`
- Site config: `basic-flow-tutorial/app.mjs`
- Header styling: `basic-flow-tutorial/app.styles.scss`
- Images: `basic-flow-tutorial/images/`
